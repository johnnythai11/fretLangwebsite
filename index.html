Fretlang

A guitar-driven programming language built with textX and Python.

Examples Mapping Architecture

Purpose and Philosophy

Fretlang turns guitar performance into code. Chords become keywords, notes become letters and numbers, and guitar techniques become syntax actions. The goal is to make programming feel musical, physical, and performative.

Mapping Overview

Chord Keywords

  • Cmaj -> let
  • Amin -> loop
  • Gmaj -> if
  • G7 -> elif
  • Emin -> print
  • Fmaj -> end or close block

Technique Actions

  • double strum -> {
  • muted strum -> }
  • slide up -> ++
  • slide down -> --
  • hammer-on -> +
  • pull-off -> -
  • bend -> *
  • harmonic -> new line

Implementation

1. Token Mapper

Maps note, chord, and technique events into Fretlang tokens.

2. textX Grammar

Parses normalized `.fret` programs using a custom DSL grammar.

3. Python Runtime

Interprets the AST and executes variables, loops, conditionals, and print.

Program Examples

print Hello
let i = 1
loop 4 {
    print i
    i++
}
let i = 1
loop 15 {
    if i % 15 == 0 {
        print FizzBuzz
    } elif i % 3 == 0 {
        print Fizz
    } elif i % 5 == 0 {
        print Buzz
    } elif i % 1 == 0 {
        print i
    }
    i++
}
